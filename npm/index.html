Installation #

  • NOT via node installer!
  • install node version manager instead nvm
  • then run "nvm install node"
  • npm install -g npm

Commands #

Check versions

npm list -g --depth=0
npm list -g @vue/cli

Find out which packages need to be updated

npm outdated -g --depth=0

update package

npm update -g <package>
